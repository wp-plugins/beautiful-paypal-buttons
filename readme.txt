Beautiful paypal buttons is an awesome easy plugin for insert paypal payment button in page, post or in widget. 


== Description ==


Beautiful paypal buttons provides you a settings panel so you can generate payment buttons easily. No need to go to paypal website & generate button code & insert ugly HTML codes in your code editor. 

Beautiful Paypal buttons is your solutions for generate payment button easily. You can generate unlimited button with custom styles. Checkout its features

<ul>
	<li>Super easy Installation. </li>
	<li>Unlimited accordions & colors. </li>
	<li>Responsive for all devices.</li>
	<li>All Major browser supported. </li>
	<li>Mobile, tablet touch supported.</li>
	<li>Minimalist & lightweight.</li>
	<li>No need to go paypal website. Just add your product name, email & other info. Your button will generate.</li>
</ul>

See live demo here: http://wpexpand.com/plugins/beautiful-paypal-buttons/



== Installation ==

Installing this plugin as regular wordpress plugin.

After install, you are ready to generate payment button. Go to post editor (Make sure you selected visual editor), Click to paypal button, you 'll see a generator. Fill with your info, you'll get a shortcode in  your editor. Now use it.
